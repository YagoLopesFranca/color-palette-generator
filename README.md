# 🎨 Color Palette Generator

Uma aplicação web responsiva desenvolvida para gerar paletas de cores de forma rápida e prática.  
O objetivo do projeto foi trabalhar lógica em JavaScript, manipulação de DOM e melhorar a experiência do usuário em uma ferramenta visual simples e funcional.

## 🚀 Funcionalidades

- Geração automática de paletas de cores aleatórias
- Atualização dinâmica das cores em tempo real
- Cópia do código HEX com apenas um clique
- Feedback visual ao copiar a cor
- Interface moderna e responsiva
- Layout organizado e focado em usabilidade

---

## 💻 Tecnologias utilizadas

Esse projeto foi desenvolvido utilizando:

- HTML5
- CSS3
- JavaScript (puro)
- Manipulação de DOM
- Eventos de clique
- Clipboard API

---

## 🎯 Objetivo do projeto

Esse projeto foi criado com foco em prática real de desenvolvimento front-end, principalmente:

- melhorar a lógica em JavaScript
- trabalhar manipulação de elementos na tela
- criar interações reais com o usuário
- desenvolver um projeto visual do zero

---

## 📱 Responsividade

A aplicação foi desenvolvida pensando em diferentes tamanhos de tela, funcionando bem em:

- Desktop

- Mobile
